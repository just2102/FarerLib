FarerLib Documentation

Endpoints

    Books

  • GET/books (get all books)
  • POST/books (post new book)

    Expects an object like this:

    {title:string,

    author:id(example 63e2444dedb5783bf04e57cd),

    genre:string,

    year?:number,

    summary?:string}
  • GET/books/bookId (get book by bookId)
  • DELETE/books/bookId (delete book by bookId)

    Authors

  • GET/authors (get all authors and their books)
  • POST/authors (post new author)

    Expects an object like this:

    {first_name: string,

    last_name: string,

    date_of_birth?: Date,

    date_of_death?: Date}

  • GET/authors/authorId (get author by authorId)
